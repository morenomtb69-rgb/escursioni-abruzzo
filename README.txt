PUBBLICAZIONE RAPIDA SU NETLIFY

1. Apri https://app.netlify.com
2. Trascina l'intera cartella 'escursioni_netlify_v2'
3. Dopo il primo deploy, vai nel sito su Netlify > Forms
4. Se richiesto, abilita Form detection
5. Da quel momento il modulo 'carica-foto-escursione' raccoglie davvero le submissions

NOTE IMPORTANTI
- Il pulsante Facebook è già presente ma il link è impostato su '#': sostituiscilo con il tuo profilo reale.
- Il caricamento foto ora usa Netlify Forms con file upload.
- Le submissions si vedono nel pannello Netlify del sito.
- Il GPX è incluso nella cartella e scaricabile.
